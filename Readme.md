# Backroad Tour Company

View Live: [backroad-tour](https://backroad-tour-test.netlify.app/)

A polished one-page landing site for a travel company, built with HTML, CSS, and a small amount of JavaScript. The project highlights featured tours, services, gallery highlights, and a newsletter signup in a clean, responsive layout.

## Highlights

- Professional hero section with a clear call to action
- About, services, featured tours, and gallery sections
- Responsive navigation and footer experience
- Newsletter form and dynamic footer year
- Smooth scroll through sections

## Run locally

Open the project folder in a browser, or serve it from a simple local server such as Live Server.
